## 📊 项目简介

**ChatBI** 是一个革命性的商业智能工具，让你通过**自然语言对话**就能快速创建专业的数据可视化大屏。无需编写代码，无需拖拽组件，只需像聊天一样描述你的需求。

### ✨ 核心特性

- 🗣️ **对话式交互**：通过自然语言描述你的数据可视化需求
- 🎨 **智能生成大屏**：AI 自动推荐合适的图表类型和布局
- 📈 **实时数据接入**：支持多种数据源（数据库、Excel、API等）
- 🔄 **迭代优化**：可通过对话持续调整和优化大屏效果
- 💾 **一键导出**：支持导出为 HTML、图片或分享链接

### 🎯 使用场景

```
用户输入 → AI理解需求 → 数据映射 → 图表推荐 → 大屏生成 → 对话微调
```
